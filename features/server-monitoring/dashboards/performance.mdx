This dashboard gives you a quick overview of the hosts's system resources and details.

You'll be able to see metrics on the load average, CPU usage, memory usage, disk usage, network traffic, and more.

## Detailed metrics

For more detailed metrics of disk usage, network traffic, see their respective dashboards.
