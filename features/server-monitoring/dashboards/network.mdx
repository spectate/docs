---
title: 'Network dashboard'
description: 'Quick insights into the network traffic of your host.'
---

This dashboard shows the network traffic of your host.

You can see metrics like:
 - Bytes received
 - Bytes sent
 - Packets received
 - Packets sent
 - Network errors
 - Dropped packets
 - FIFO buffer errors
