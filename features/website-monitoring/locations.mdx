---
title: Locations
---

Spectate provides you with multiple locations which you can use to monitor your application from. You can choose between the following locations:

* Amsterdam
* Bangalore
* Falkenstein
* London
* New York
* San Francisco
* Singapore
* Sydney

More locations will be added in the future.

<Tip>
    You can find the IP addresses of the location [here](/faq/ip-addresses). This is useful if you have to whitelist the IP addresses of the Spectate servers.
</Tip>
