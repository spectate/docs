## Setup

1. Go to [Integrations](https://app.spectate.net/integrations) and click on `Configure` in the Telegram card.
2. Manually invite `@SpectateAlertBot` to your chat or group or click the button to open the chat with the bot.
3. Run the provided `/start@SpectateAlertBot` command in the chat with the bot.
4. Click `Refresh`.
5. Go to `Configurations` and select your newly created configuration.
6. You can now send a test notification to make sure everything is working.
