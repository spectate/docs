# Tags

Tags can be used to organize your monitors. They can be applied to monitors and can be used to filter monitors in the monitor list.

## Create a tag

Simply go to the monitor's settings and start typing in the tags field. If the tag does not exist, it will be created automatically.

## Filter monitors by tag

You can filter monitors by tag by just using the search field in the monitor list. Simply type in the tag name and all monitors with that tag will be shown.

## Notification channels

Tags are not only useful for filtering monitors, but also for sending notifications to different channels. You can create a tag and then add notification channels to it. When a monitor with that tag triggers, the notification will be sent to the channels that are associated with the tag.
