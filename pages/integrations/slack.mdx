# Slack

Get notified about new incidents directly in your companies' Slack.

## Setup

1. Go to the [Spectate Incident Channels](https://app.spectate.net/incidents/channels) and select "Slack".
2. Click "Connect Slack workspace" and follow the instructions.
3. Once installed, enter the name of the channel you want to use for incident notifications.
4. You can now send a test notification to make sure everything is working.
