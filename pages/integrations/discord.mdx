# Discord

Get notified about new incidents directly in your Discord server.

## Prerequisites

First, you have to create a webhook in your Discord server. You can find a guide on how to do that [here](https://support.discord.com/hc/en-us/articles/228383668-Intro-to-Webhooks).

## Setup

1. Go to the [Spectate Incident Channels](https://app.spectate.net/incidents/channels) and select "Discord".
2. Paste your webhook URL and click "Add Channel".
3. You can now send a test notification to make sure everything is working.
