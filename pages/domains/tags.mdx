# Tags

Tags can be used to organize your domain names. They can be applied to domain names and can be used to filter domain names in the list.

## Create a tag

Simply go to the domain name's settings and start typing in the tags field. If the tag does not exist, it will be created automatically.

## Filter domain names by tag

You can filter domain names by tag by just using the search field in the list. Simply type in the tag name and all domain names with that tag will be shown.

## Notification channels

Tags are not only useful for filtering monitors, but also for sending notifications to different channels. You can create a tag and then add notification channels to it. When a domain names with that tag triggers, the notification will be sent to the channels that are associated with the tag.
